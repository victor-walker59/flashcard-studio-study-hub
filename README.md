## What is flashcard studio?

flashcard studio is a web-first flashcard tool made for simple study routines. You can build cards in the browser, come back to them for review, and use the same app for quick quiz sessions without installing a desktop application.

It suits students, self-learners, and anyone who wants a compact study utility that lives in a browser tab. The design stays centered on creating cards and repeating them, which makes it practical for small practice decks as well as ongoing review.

---

## Key capabilities

- Build flashcards straight in an HTML browser interface
- Go through cards using a study flow built for repetition
- Launch quiz-style practice from the same workspace
- Use the app without a native desktop install
- Study in supported browsers within a web environment
- Keep the experience centered on flashcard learning
- Transition from card creation to review with very few steps

## How to use it

A common workflow looks like this:

1. Start the app in your browser.
2. Add one or more flashcards with the material you want to learn.
3. Review the cards to strengthen recall.
4. Run quiz-style practice to check memory.
5. Come back later and repeat the cycle whenever needed.

Example local workflow with a simple server:

`python -m http.server`

Then visit the local address shown in your terminal and load the app from there.
